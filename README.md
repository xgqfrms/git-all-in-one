## Basic Git

### help

```sh
    
# help
$ git -h
$ git --help


```

### checkout

```sh
    
# add
$ git checkout


```

### add

```sh
    
# add
$ git add .


```

### commit

```sh
    
# commit
$ git commit -m "commit comments"


```

### status

```sh    
# status
$ git status


```

## git config all in one


https://github.com/xgqfrms/git/issues


```sh
# global config
$ git config
$ git config --global


# list
$ git config -l

# edit
$ git config -e

# vscode
$ git config --global core.editor vscode

```

```sh
# local config
$ git config --local -l
# local edit
$ git config --local -e

```
